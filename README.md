# Задание 1

## Задача 1

### Текст задачи

Решить задачу на построение нового дерева по заданному (map). Дерево содержит целые числа. Увеличить каждый разряд на 1, если он не содержит 9, иначе сохранить разряд неизменным. Например, 1296 ➔ 2397

### Алгоритм решения

Рандомно генерируем дерево: с помощью генератора случайных чисел рекурсивно генерируем узлы дерева. Так же нужно учесть ограничение на максимальную длину дерева, чтобы не получить бесконечную рекурсию. Сами значения узлов тоже генерируются случайно. Выводим полученное дерево пользователю. Затем применяем ко всему дереву рекурсивную функцию преобразования разрядов: если аргумент функции меньше 10 - возвращаем преобразованное согласно правилу число. Если аргумент функции был больше 10 - отделяем последний разряд от числа, преобразуем его, складываем с результатом рекурсии от оставшейся части числа. Выводим новое дерево пользователю

# Задание 2

## Задача 1

### Текст задачи

Решить задачу на получение заданного результата для указанного дерева (fold) Дерево содержит строки. Сколько узлов дерева не содержит заданный символ?

### Алгоритм решения

Рандомно генерируем дерево: с помощью генератора случайных чисел рекурсивно генерируем узлы дерева. Так же нужно учесть ограничение на максимальную длину дерева, чтобы не получить бесконечную рекурсию. Сами значения узлов тоже генерируются случайно: в самом коде прописываем строку символов. Случайным образом определяем длину итоговой строки. Создаем массив символов с помощью генератора с циклом и нашей строкой символов. Преобразуем массив символов в строку. Выводим дерево пользователю. Запрашиваем символ у пользователя. Рекурсивно проходим по дереву, если узел содержит символ - возвращаем 0, если нет символа - возвращаем 1 + значения этой же функции для дочерних узлов. Выводим результат
