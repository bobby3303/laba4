# Задание 1

## Задача 1

### Текст задачи

Решить задачу на построение нового дерева по заданному (map). Дерево содержит целые числа. Увеличить каждый разряд на 1, если он не содержит 9, иначе сохранить разряд неизменным. Например, 1296 ➔ 2397

### Алгоритм решения

Рандомно генерируем дерево: с помощью генератора случайных чисел определяем количество узлов дерева, значения узлов. Рекурсивно создаем дерево, вставляя значения узлов согласно бинарному дереву поиска. Выводим полученное дерево пользователю. Затем применяем ко всему дереву функцию map, аргументами которой являются дерево и  функция преобразования разрядов: если аргумент функции меньше 10 - возвращаем преобразованное согласно правилу число. Если аргумент функции был больше 10 - отделяем последний разряд от числа, преобразуем его, складываем с результатом рекурсии от оставшейся части числа. Выводим новое дерево пользователю.

# Задание 2

## Задача 1

### Текст задачи

Решить задачу на получение заданного результата для указанного дерева (fold) Дерево содержит строки. Сколько узлов дерева не содержит заданный символ?

### Алгоритм решения

Рандомно генерируем дерево: с помощью генератора случайных чисел определяем количество узлов дерева. Сами значения узлов тоже генерируются случайно: в самом коде прописываем строку символов. Случайным образом определяем длину итоговой строки. Создаем массив символов с помощью генератора с циклом и нашей строкой символов. Преобразуем массив символов в строку. Из полученных строк формируем бинарное дерево поиска. Выводим дерево пользователю. Запрашиваем символ у пользователя. Применяем функцию fold, аргументами которой являются функция подсчета строк с уже вставленным символом, начальное значение аккумулятора, и само дерево. Выыводим результат.
